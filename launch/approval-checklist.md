# Public Posting Approval Checklist

Use this before any public post, submission, DM batch, or community comment.

## Required

- [ ] The post clearly says the owner built or is working on Cash Runway.
- [ ] The post says it is free if relevant.
- [ ] The post does not claim guaranteed financial results.
- [ ] The post does not give individual financial advice.
- [ ] The post does not imply bank/security integrations that do not exist.
- [ ] The post does not say data is encrypted in cloud storage because there is no cloud storage.
- [ ] The post does not hide that data is stored locally in the browser.
- [ ] The community rules were checked.
- [ ] The post is customized to the community.
- [ ] The post asks for feedback or shares a useful lesson, not just traffic.
- [ ] The owner approved the exact final text and destination.

## Risk Flags

Do not post without revising if the copy:

- Sounds like an ad.
- Sounds like financial advice.
- Promises overdraft prevention.
- Mentions "guaranteed", "risk-free", or "will save you money".
- Repeats the same message across many communities.
- Targets people in distress with pressure language.

## Safer Language

Use:

- "Forecast"
- "Planning tool"
- "Can help you see"
- "May help identify"
- "Not financial advice"
- "Local-only data"

Avoid:

- "Guarantee"
- "Prevent overdrafts"
- "Fix your finances"
- "Best financial app"
- "Bank-grade"
